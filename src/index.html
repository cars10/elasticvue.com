Usage

Chose how you want to run elasticvue

Web app

Elasticvue is available online:

Open app.elasticvue.com

Docker

Download and run the image:

docker run -p 8090:8090 \
           -d cars10/elasticvue

Then open http://localhost:8090 in your browser.

Chrome extension

Elasticvue is available as chrome extension:

Open chrome webstore

Start elasticvue by clicking on the icon in your toolbar.

Checkout the github repository if you want to build elasticvue by yourself.

Then procced to the configuration

Configuration

Configure your elasticsearch server

You have to enable CORS for your elasticsearch server to allow access from external domains.

Add the following to your elasticsearch configuration, based on how you run elasticvue: 

# Enable 'Access-Control-Allow-Origin' header
http.cors.enabled: true

# Then set the allowed origins based on how you run elasticvue:

# for docker / running locally
http.cors.allow-origin: /https?:\/\/localhost(:[0-9]+)?/
# online version
http.cors.allow-origin: /https?:\/\/elasticvue.cars10k.de/
# chrome extension
http.cors.allow-origin: /chrome-extension:\/\/[a-z]+/

Now simply restart elasticsearch and you should be able to connect.
