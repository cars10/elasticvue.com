Elasticvue is a free and easy to use frontend for elasticsearch.

Features

  • Cluster overview
  • Index management
  • Search interface
  • Query interface (REST & API browser)
  • Snapshot management
  • Utilities

Usage

Usage

There are several ways of running elasticvue, chose what works best for you:

Web app

Elasticvue is available online:

Open app.elasticvue.com

This is the easiest way to use elasticvue.

Docker image

Download and run the image:

docker run -p 8080:8080 \\
           -d cars10/elasticvue

Then open http://localhost:8080 in your browser.

Chrome extension

Elasticvue is available as a chrome extension:

Open chrome webstore

Start elasticvue by clicking on the icon in your toolbar.

Or use the firefox addon

Firefox addon

Elasticvue is available as a firefox addon:

Open firefox addon

Start elasticvue by clicking on the icon in your toolbar.

Or use the chrome extension

Cluster configuration

Configuration

Configure your elasticsearch server

To use elasticvue you have to enable CORS in your elasticsearch server. This will allow access across different origins, otherwise your browser will reject all requests because they are marked as insecure.

To enable CORS add the following to your elasticsearch configuration, based on how you run elasticvue:

  • http.cors.enabled: true
  • http.cors.allow-origin: <your-elasticvue-origin>

After configuration restart elasticsearch and you should be able to connect.

Configuration example
# Enable CORS
http.cors.enabled: true

# Then set the allowed origins based on how you run elasticvue.
# Chose only one:

# for docker / running locally
http.cors.allow-origin: "http://localhost:8080"
# online version
http.cors.allow-origin: /https?:\\/\\/app.elasticvue.com/
# chrome extension
http.cors.allow-origin: "chrome-extension://hkedbapjpblbodpgbajblpnlpenaebaa"
# firefox addon
http.cors.allow-origin: "moz-extension://4b05c928-89bb-4a48-af52-b871f123a06b"

# and if your cluster uses authorization you also have to add:
http.cors.allow-headers : X-Requested-With,Content-Type,Content-Length,Authorization
