Elasticvue features

See elasticvue in action.

${require('./components/edge_bottom_left.html')}

Cluster overview

Elasticvue displays the response of both endpoints for node information and the whole cluster health.

Node overview

You can view every node in your cluster in a table or grid view.

Index management

You can manually create and manage your indices. This includes viewing index settings and stats or opening or closing the index.

Document search

You can search documents and view the results in a fully featured table, including sorting, filtering and pagination.

REST gui

Elasticvue has a gui to query your cluster via direct HTTP requests.
Alternatively you can use the api browser and select every available elasticsearch api method directly.

Snapshot & repositories

Manage repositories and snapshots right in your browser.
You can create, delete and restore snapshots based on index patterns.
